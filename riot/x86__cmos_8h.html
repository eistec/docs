void x86_cmos_serial ( uint8_t(*)  serial[CMOS_SERIAL_LEN])

Read serial number of the BIOS.

Most – if not all current – BIOSes do not seem to store any serial number here. This function won't cause an error, but the read serial might be some more or less random data. The implementor of the board specific code should know whether the BIOS contains a serial number.
