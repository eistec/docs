Detailed Description

Driver for the LM75A digital temperature sensor and thermal watchdog.

The connection between the MCU and the LM75A is based on the i2c-interface.

bool lm75A_init ( uint8_t  i2c_interface,
uint32_t  baud_rate,
void *  external_interr_handler 
)

Initialize the LM75A temperature sensor.

The baud rate and the handler for the external interrupt can be initialized. The external interrupt handler is optional, if no handler is available, the NULL-value can be entered. The hysteresis and the over-temperature are displayed before and after a rest action is performed. After this the LM7A sensor is set in the interrupt or the comparator mode.

Parameters
[in]i2c_interfacethe i2c interface, several interfaces can be selected: i2c0, i2c1 and i2c2.
[in]baud_ratethe baud rate.
[in]external_interr_handlerpointer to a handler for the external interrupt.
Returns
true if the I2C interface and the external interrupt handler are successfully initialized, otherwise false.
