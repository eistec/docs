#define PURE

Variable Documentation

volatile int lpm_prevent_sleep

This bitfield is used to configure which modules are currently active and prevent the kernel to go to the deepest power modes.

It is used with LPM_PREVENT_SLEEP_HWTIMER and/or LPM_PREVENT_SLEEP_UART.
