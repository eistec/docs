Typedef Documentation

typedef struct pthread_barrier pthread_barrier_t

A synchronization barrier.

Initialize with pthread_barrier_init(). For a zeroed out datum you do not need to call the initializer, it is enough to set pthread_barrier_t::count.

typedef struct pthread_barrierattr pthread_barrierattr_t

Details for a pthread_barrier_t.

RIOT does not need this structure, because it is a single process OS. This is only here to POSIX compatibility.

int pthread_barrier_destroy ( pthread_barrier_t *  barrier)

Destroys a pthread_barrier_t.

To use the barrier again you will need to call pthread_barrier_init() again. Destroying a barrier while threads are currently waiting for it causes indefined behavior.
