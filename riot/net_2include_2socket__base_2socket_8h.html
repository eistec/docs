int  type,

)

Creates new socket for communication in family domain, of type type, and with protocol protocol.

Roughly identical to POSIX's socket(2).

Parameters
[in]domainAddress family. Only AF_INET6 is supported by now (no check on creation).
[in]typeCommunication type.
