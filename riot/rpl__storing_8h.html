Detailed Description

Routing Protocol for Low power and Lossy Networks.

RPL storing-mode header

Header which includes all mode related RPL-functions. All functions are mandatory for any RPL-mode. Describes receiving and sending of all RPL-related messages and special initialization behavior.

void rpl_send ( ipv6_addr_t *  destination,
uint8_t *  payload,
uint16_t  p_len,
uint8_t  next_header 
)

Sends a RPL message to a given destination.

This function sends any RPl related messages to a given destination. This implementation should be equal for all modes and therefore should not be altered. Every mode related RPL-sending function calls this for relaying it in lower layers to sixlowpan. Because send-functions are wrapped by a mutex in rpl.c, the same mutex applies here.

Parameters
[in]destinationIPv6-address of the destination of the message.
[in]payloadPayload of the message.
[in]p_lenLength of the message
[in]next_headerIndex to next header in message.
