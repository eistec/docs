Detailed Description

Interface definition for the global network buffer.

Network devices and layers can allocate space for packets here.

Note
WARNING!! Do not store data structures that are not packed (defined with __attribute__((packed))) or enforce alignment in in any way in here. On some RISC architectures this will lead to alignment problems and can potentially result in segmentation/hard faults and other unexpected behaviour.
