void ipv6_iface_set_routing_provider ( ipv6_addr_t *(*)(ipv6_addr_t *dest)  next_hop)

Registers a function that decides how to route incoming IP packets with a destination that is not this interface.

The default behavior is to try forwarding such packets to the neighborhood. Register a function to change the default behavior. Such function shall return the next hop to reach the destination of the IP packet, or NULL if no such next hop is known. In this case, the packet will be discarded.

Parameters
next_hopfunction that returns the next hop to reach dest

Tries to determine best suitable source address attached to an interface of this node based on the given destination address.

The use-case for this function is to find a suitable address for the source address field of an IPv6 address upon sending. src may be empty (all zero) if there is no suitable address attached to the interface.

Parameters
