x86_intr_handler_t  handler 
)

Set callback function for interrupt.

Parameters
numNumber of interrupt. Must be less than X86_MAX_INTERRUPTS.
handlerFunction to call, or NULL to use default interrupt handler.

This function is meant to be called by other subsystems (x86_pic.c and x86_threading.c). Any upper level system should use x86_pic_set_handler() or even higher level functions.
