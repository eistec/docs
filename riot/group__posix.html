int close ( int  fildes)

Close a file descriptor.

shall deallocate the file descriptor indicated by fildes. To deallocate means to make the file descriptor available for return by subsequent calls to open() or other functions that allocate file descriptors. All outstanding record locks owned by the process on the file associated with the file descriptor shall be removed (that is, unlocked).

See also
The Open Group Base Specification Issue 7, close
Parameters
[in]fildesThe file descriptor to the file which is to close.
Returns
Upon successful completion, 0 shall be returned; otherwise, -1 shall be returned and errno set to indicate the error.

unsigned int sleep ( unsigned int  seconds)

the caller will sleep for given amount of seconds

The sleep() function shall cause the calling thread to be suspended from execution until either the number of realtime seconds specified by the argument seconds has elapsed or a signal is delivered to the calling thread and its action is to invoke a signal-catching function or to terminate the process. The suspension time may be longer than requested due to the scheduling of other activity by the system.

See also
The Open Group Base Specification Issue 6, sleep
