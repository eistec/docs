void videoram_putc ( char  c)

Print out a single character on the graphical device.

This method can be used before the UART system was initialized.

Special characters that are understood: '\n' and '\r'. Use DOS newlines.

void videoram_puts ( const char *  s)

Print out multiple characters on the graphical device, then go to the next line.

This is the same as calling videoram_put(s); videoram_put("\\r\\n");.
