bool cc1100_set_mode ( uint8_t  mode,
uint16_t  opt_mode_data 
)

Set the radio mode.

Sets the radio mode with optional mode data. If the radio mode is WOR, the optional mode data is the RX interval in milliseconds. Must be called only after CC1100 is initialized.

Parameters
modeNew radio mode (one of the constants defined in this file).
opt_mode_dataDesired RX interval in milliseconds [50..60000].
Returns
true if mode could be applied; false otherwise (stays in previous mode).

bool cc1100_set_packet_monitor ( packet_monitor_t  monitor)

Set a packet monitor at this layer.

All incoming packets will be passed to this packet monitor before packet gets processed by a specific handler. The work in the monitor should take as less time as possible (no waiting or intense I/O).

Parameters
monitorThe packet monitor or NULL if the current packet monitor should be removed.
Returns
true if this layer supports packet monitoring; false otherwise.
