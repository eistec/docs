Initialization of a given DAC device.

The DAC will be initialized with all possilble channels active. On each channel will be initialized with a Zero on it.

Parameters
[in]devthe device to initialize
[in]precisionthe precision to use for conversion
Returns
0 on success
-1 on unknown DAC Device
-2 on precision not available

int  value,
int  min,

)

Helper function to map a given integer range to a valid DAC value.

This function is useful for converting ranges of values to a valid DAC output value.

The min value is assumed to be smaller than max value and value is assumed to be between min and max.

Parameters
[in]devthe DAC Device to read precision value from
[in]valuethe value to map onto min and max
[in]minthe lower bound of the target interval
[in]maxthe upper bound of the target interval
Returns
the mapped value, in valid DAC range
