Detailed Description

IEEE 802.15.4 radio driver API definition.

This is the set of functions that must be implemented by any driver for a 802.15.4 radio transceiver.

int(* netdev_802154_driver_t::add_receive_raw_callback)(netdev_t *dev, netdev_802154_raw_packet_cb_t recv_func)

Add a function to be called back when the radio transceiver has received a incoming packet.

This function differentiates from netdev_driver_t::add_receive_data_callback() as it expects a callback that excepts the raw frame format of IEEE 802.15.4, rather than just the source and destination addresses and the payload.

Parameters
[in]devthe network device
[in]recv_functhe callback function to invoke for each packet received by the radio transceiver.
See also
netdev_802153_raw_packet_cb_t
Returns
0, on success
-ENOBUFS, if maximum number of registrable callbacks is exceeded
-ENODEV, if dev is not recognized

netdev_802154_tx_status_t(* netdev_802154_driver_t::load_tx)(netdev_t *dev, netdev_802154_pkt_kind_t kind, netdev_802154_node_addr_t *dest, int use_long_addr, int wants_ack, netdev_hlist_t *upper_layer_hdrs, void *buf, unsigned int len)

Load the transceiver TX buffer with the given IEEE 802.15.4 packet.

Parameters
[in]devthe network device
[in]kindKind of packet to transmit.
[in]destAddress of the node to which the packet is sent.
[in]use_long_addr1 to use the 64-bit address mode with dest param; 0 to use "short" PAN-centric mode.
[in]wants_ack1 to request an acknowledgement from the receiving node for this packet; 0 otherwise.
[in]upper_layer_hdrsheader data from higher network layers from highest to lowest layer. Must be prepended to the data stream by the network device. May be NULL if there are none.
[in]bufPointer to the buffer containing the payload of the 802.15.4 packet to transmit. The frame header (i.e.: FCS, sequence number, src and dest PAN and addresses) is inserted using values in accord with kind parameter and transceiver configuration.
[in]lenLength (in bytes) of the outgoing packet payload.
Returns
The outcome of this packet's transmission.

netdev_802154_tx_status_t
